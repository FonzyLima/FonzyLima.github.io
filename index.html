Personal Discord Bot

Vax-in Mobile Application


Vax-in is an Android Application created in fulfillment to my Mobile Application Development course (MOBDEVE). It is an application that would allow users to register for a vaccine and see their overall progress in the vaccination process. This app also has an admin side that handles the scheduling and managing of users.

I handled the overall back-end development of this application which includes the usage of the Firebase Realtime Database, and the SmsManager Package. This application was made using Android Studio and was coded in Java.

Tic-tac-toe Expert System


This program simulates a simple Tic-Tac-Toe game against an artificial intelligence with three levels of difficulty. The first level of difficulty makes the AI do random moves except when a winning move is possible. The second level of difficulty focuses more on initial placements and blocking out a user's winning moves. The last level of difficulty is the expert level which makes all the right moves that leads to the AI's most optimal position. It is impossible to win against the expert level AI. It can only lead to a draw or a loss for you.

The Minimax Algorithm was used to design the expert system of the program. The program was coded in Python with the use of the Pygame and Tkinter packages for the user interface.
